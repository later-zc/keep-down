# 驾驶证实习期

机动车驾驶人初次取得汽车类准驾车型或初次取得摩托车类准驾车型后的**十二个月为实习期**，在实习期内驾驶机动车的，
应当在机动车后部黏贴或悬挂统一式样的实习标志。



# 实习期驾驶车辆要求

机动车驾驶人在实习期内，不得驾驶公共汽车、营运客车、或者执行任务的警车、消防车、救护车、工程救险车、以及载有爆炸物品、易燃易爆化学物品、
剧毒、放射性等危险物品的机动车。

驾驶的机动车不得是牵引挂车。



# 实习期上路行驶要求

驾驶人**在实习期内驾驶机动车上高速公路行驶应当由持相应或包含其准驾车型驾驶证3年以上的驾驶人陪同**，其中驾驶残疾人专用小型自动挡载客汽车的，
可以由持小型自动挡载客汽车以上准驾车型驾驶证的驾驶人陪同。

在增加准驾车型后的实习期内，驾驶原准驾车型的机动车时，不受上述限制，在实习期内违法上述规定，驾驶机动车的，由公安机关交通管理部门，处20-200元罚款。

持有准驾车型为残疾人专用小型自动挡载客汽车的机动车驾驶人，驾驶机动车时应当按规定，在车身设置残疾人机动车专用标志，有听力障碍的机动车驾驶人，
在驾驶机动车时，应当佩戴助听设备驾驶，有视力矫正的机动车驾驶人驾驶机动车时，应当佩戴眼镜。

::: warning
遇到实习车辆，应当提前注意实习车辆动向，或礼让。

实习期内的驾驶人，发生道路交通安全违法行为，被记满12分的，将被注销其实习的准驾车型驾驶资格。
:::



# 驾驶证换证与补证

驾驶证的有效期分为6年、10年和长期三种类型。每个记分周期未达到12分的，可以换发年限更高的驾驶证。

## 发证要求

申请人考试合格后，应当接受不少于半小时的交通安全文明驾驶常识和交通事故案例警示教育，并参加领证宣誓仪式。

车辆管理所应当在申请人参加领证宣誓仪式的当日，核发机动车驾驶证。

::: warning
申请人属于复员、转业、退伍人员持军队、武装警察部队机动车驾驶证，申请机动车驾驶证的，应当收回军队、武装警察部队机动车驾驶证。

此外，申请人以欺骗、贿赂等不正当手段取得机动车驾驶证的，由公安机关交通管理部门收缴机动车驾驶证、撤销机动车驾驶证许可，处2000元以下罚款，
申请人在三年内不得再次申领机动车驾驶证。

组织、参与实施以下行为之一，牟取经济利益的，由公安机关交通管理部门，处违法所得3-5倍罚款，但最高不超过10万元。

1. 隐瞒有关情况或提供虚假材料申领机动车驾驶证的；
2. 考试过程中有贿赂、舞弊行为的；
3. 以欺骗、贿赂等不正当手段取得驾驶证的；

申请人隐瞒或提供虚假材料申请校车驾驶资格的，公安机关交通管理部门不予受理或不予办理，处500元以下罚款，申请人一年内不得再次申请校车驾驶资格。

申请人以欺骗、贿赂等不正当手段取得校车驾驶资格的，公安机关交通管理部门撤销校车驾驶资格，处2000元以下罚款，申请人三年内不得再次申请校车驾驶资格。
:::

## 驾驶证电子化要求

公安机关交通管理部门，应当实行机动车驾驶证电子化，机动车驾驶人可以通过互联网交通安全综合服务管理平台，申请机动车驾驶证电子版。

机动车驾驶证电子版与纸质版具有同等效力。

## 驾驶证有效期满换证要求

在机动车驾驶证的6年有效期内，每个记分周期均未满12分的，换发10年有效期的机动车驾驶证；
在机动车驾驶证的10年有效期内，每个记分周期均未满12分的，换发长期有效期的机动车驾驶证；

机动车驾驶人应当于机动车驾驶证有效期满前90日内，向机动车驾驶证核发地或核发地以外的车辆管理所申请换证，申请时应当确认申请信息
并提交以下证明、凭证：
1. 机动车驾驶证的身份证明
2. 医疗机构出具的有关身体条件的证明

::: warning 
年龄在70周岁以上的机动车驾驶人，应当每年进行一次身体检查，在记分周期结束后30日内，提交医疗机构出具的有关身体条件的证明。
:::

## 驾驶证户籍迁出换证要求

机动车驾驶人 户籍迁出原车辆管理所管辖区的，应当向迁入地车辆管理所申请换证。

机动车驾驶人 在核发地车辆管理所管辖区以外居住的，可以向居住地车辆管理所申请换证。

转入换证时，应当确认申请信息，提交机动车驾驶人的身份证明和机动车驾驶证，并申报身体条件情况，车辆管理所应当在1日内换发机动车驾驶证，并收回原机动车驾驶证。

## 驾驶证信息变更或损毁换证要求

在车辆管理所管辖区域内，机动车驾驶证记载的机动车驾驶人信息发生变化的，或是机动车驾驶证损毁无法辨认的，
机动车驾驶人应当在30日内，到机动车驾驶证核发地或核发地以外的车辆管理所申请换证，申请时，应当确认申请信息，
并提交机动车驾驶人的身份证明，
> 属于在车辆管理所管辖区域内，机动车驾驶证记载的机动车驾驶人信息发生变化的，的还应当提交机动车驾驶证。
> 
> 属于身份证号码变更的，还应当提交相关变更证明

车辆管理所应当在1日内换发机动车驾驶证，并收回原机动车驾驶证。

情形一：年龄在63周岁以上的，不得驾驶大型客车、重型牵引挂车、城市公交车、中型客车、大型货车、轮式专用机械车、无轨电车、有轨电车。
持有大型客车、重型牵引挂车、城市公交车、中型客车、大型货车驾驶证的，应当到驾驶证核发地或核发地以外的车管所，换领准驾车型为
小型汽车或小型自动挡汽车准驾车型的驾驶证。
其中持有重型牵引挂车驾驶证的，还可以保留轻型牵引挂车准驾车型，车管所应当1日内换发驾驶证，并收回原驾驶证。

情形二：年龄在70周岁以上的，不得驾驶低速载货汽车、三轮汽车、轻型牵引挂车、普通三轮摩托车、普通二轮摩托车，持有普通三轮摩托车、
普通二轮摩托车驾驶证的，应当到驾驶证核发地或核发地以外的车管所，换领准驾车型为轻便摩托车的驾驶证。
持有驾驶证包含：轻型牵引挂车准驾车型的，应当到驾驶证核发地或核发地以外的车管所，换领准驾车型为小型汽车或小型自动挡汽车准驾车型的驾驶证。
车管所应当1日内换发驾驶证，并收回原驾驶证。

有以上两种情形之一的，车管所应当通知驾驶人，在30日内办理换证业务，驾驶人逾期未办理的，车管所应当公告准驾车型驾驶资格作废。
申请时应当确认申请信息，并提交机动车驾驶人的身份证明，医疗机构出具的有关身体条件的证明。

机动车驾驶人自愿降低准驾车型的，应当确认申请信息，并提交机动车驾驶人的身份证明和机动车驾驶证，车管所应当在1日内换发机动车驾驶证，并收回原机动车驾驶证。

## 驾驶人身体条件发生变化

驾驶人身体条件发生变化，不符合所持驾驶证准驾车型条件，但符合准予驾驶的其他准驾车型条件的，应当在30日内到驾驶证核发地或核发地以外的车辆管理所申请
降低准驾车型，车管所应当在1日内换发机动车驾驶证，并收回原机动车驾驶证。申请时，应当确认申请信息，并提交机动车驾驶人的身份证明、医疗机构出具的有关身体条件的证明。

驾驶人身体条件发生变化，不符合所持驾驶证准驾车型条件，或者具有申请驾驶证的[禁止条件情形](/notes//驾考科目一理论/驾驶证的申领和使用/1.2%20驾驶证的申领#禁止申请情形)之一不适合驾驶机动车的，
应当在30日内，到驾驶证核发地的车管所申请注销，车管所应当在1日内注销该驾驶证。申请时应当确认申请信息，并提交机动车驾驶人的身份证明和机动车驾驶证。

::: warning
驾驶人身体条件不适合驾驶机动车的，不得驾驶机动车。
:::

## 驾驶证遗失补证要求

机动车驾驶证遗失的，驾驶人应当向驾驶证核发地或核发地以外的车管申请补发，申请时应当确认申请信息，并提交机动车驾驶人的身份证明。
符合规定的，车管所应当在1日内补发机动车驾驶证。

驾驶人补发驾驶证后，原驾驶证作废，不得继续使用，继续使用原驾驶证的，由公安机关交通管理部门收回原机动车驾驶证，并处20-200元罚款。

机动车驾驶证被依法扣押、扣留或暂扣期间，驾驶人不得申请补发，采用隐瞒、欺骗手段补领驾驶证的，由公安机关交通管理部门收回驾驶证，并处200-500元罚款。

::: warning
遗失、补证期间，因不符合随身携带机动车驾驶证，故不得驾驶机动车。
:::

## 驾驶证逾期换证要求

::: tip
超过换证或体检时间一年以内，驾驶证为逾期未换证状态，驾驶人可以持身份证、失效驾驶证、医疗机构出具的有关身体条件的证明，到车管所办理手续，
不用参加任何考试就可以办领证件。

驾驶证过期一年未满三年，超过换证或体检时间满一年未到三年的，在这两年期间内，驾驶证为注销可恢复状态。
驾驶人只需到车管所参加科目一考试，考试合格即可正常办理换证手续。
:::
